# `docs/assets/` — optional images

This folder is reserved for **first-party** diagrams or screenshots **you** own (export from your editor, draw.io, etc.).

**Third-party product logos** (Semgrep, Meta, Vercel, Keygraph, OpenAI, …) are **not** committed here by default: trademark and redistribution rules vary. Prefer:

- **Text links** to each vendor’s site or GitHub repo (see main README and [ARCHITECTURE_AND_FLOWS.md](../ARCHITECTURE_AND_FLOWS.md)).
- Optional **[Shields.io](https://shields.io/)** badges in markdown (small, widely used pattern) if you need a visual cue — still verify each vendor’s badge/logo policy for your context.

If you add PNG/SVG files, document the source and license in this README.
